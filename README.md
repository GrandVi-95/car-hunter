# 🚗 Car Hunter Bot

בוט שסורק יד2 ושולח התראות בטלגרם כשנמצא רכב מתאים.

## משתני סביבה ב-Railway

| משתנה | ערך |
|-------|-----|
| `TELEGRAM_TOKEN` | הטוקן מ-BotFather |
| `CHAT_ID` | המספר שלך (984548343) |
| `CHECK_INTERVAL` | 3600 (כל שעה) |

## הפעלה ב-Railway

1. צור חשבון ב-railway.app
2. לחץ "New Project" → "Deploy from GitHub"
3. העלה את הקבצים ל-GitHub repository חדש
4. הגדר את משתני הסביבה
5. הבוט יתחיל לרוץ אוטומטית!
